Linux计算器bc

功能强大

  • 基本计算器功能
  • 支持变量、函数，条件和循环 等编程 功能（类似 C 语言的小编程语言）
  • 可以进行任意精度的 计算

命令设置缺省精度

打开计算器的命令 描述
bc 缺省精度为小数点后 0 位
bc l 缺省精度 为小数点后 20 位
